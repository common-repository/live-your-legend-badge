== Description ==

Display a Live Your Legend badge in the sidebar of your blog.

There is the option to link to liveyourlegend.net.

The widget allows you to select a title, the badge you want to display (more coming soon) and whether you wish to link to livingyourlegend.net

== Installation ==

Upload the Live Your Legend Badge plugin to your blog, activate it, then add the widget (Live Your Legend Badge) on the widgets page. Configure the widget with a title, badge and link option.
